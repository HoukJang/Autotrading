## 3. 전략팀 패널 토의

### [퀀트-Q] 데이터 진단

adx_breakout(14B)과 ema_cross_trend(15B)의 실패 패턴이 동일:

| 지표 | adx_breakout (14B) | ema_cross_trend (15B) |
|------|-------------------|----------------------|
| WR | 19.4% | 16.9% |
| SL 피격률 | 76.5% | 76.3% |
| PF | 0.782 | 0.398 |

MAE(4.06%) > MFE(3.39%)가 핵심 문제. 진입 후 유리한 방향보다 불리한 방향으로 더 많이 움직임. EMA 교차 시점이 "이미 늦다"는 의미. 교차는 이미 일어난 이벤트이고, 진입 시점에 초기 모멘텀은 소진된 상태.

### [전략-T] 시장 구조적 원인

1. 전환기 시장에서 EMA 교차가 빈번하게 발생 -> 대부분 페이크아웃
2. ADX > 25가 트렌드 지속을 보장하지 않음. 잠깐 넘었다가 다시 내려오는 경우 다수
3. 교차 타이밍 문제: 교차 발생 시 이미 가격이 움직인 후라서 풀백에 노출

**근본 질문**: 이 시장(전환기)에서 트렌드 전략이 가능한가? 두 번 연속 동일 패턴(SL 76%)으로 실패.

### [리스크-R] SL/TP 구조 분석

SL 2.5 ATR, TP 4.0 ATR -> R:R = 1:1.6.
WR 16.9%에서 수익을 내려면 WR 최소 39% 필요 (1/(1+1.6) = 38.5%).
현재 WR은 이에 턱없이 못 미침.
